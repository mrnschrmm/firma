# Site - Project: Firma

`Node 12.16.1` `NPM 6.14.2` `PHP 7.3.15` `Composer 1.9.3` `Kirby 3.3.4`

## Setup
```
npm install
composer install
```

## Clone
```
npm run clone
```

## Run
```
npm run start
```

## Build
```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.full
npm run deploy.preview
npm run deploy.preview.full
```

## Debug
```
npm run start.debug
npm run build.debug
```
