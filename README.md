# Site - Project: Firma

`Node 12.15.0` `NPM 6.13.7` `PHP 7.3.14` `Composer 1.9.3` `Kirby 3.3.3`

## Setup
```
npm install
composer install
```

## Clone
```
npm run clone
```

## Run
```
npm run start
```

## Build
```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.preview
```

## Debug
```
npm run start.debug
npm run build.debug
```
